Strive Team Generator

This app will allow you to generate teams and assign the participants to them!

Features:
- Add a list of names.
- Set the number of teams you want to be generated.
- Each team will be randomly generated from the list of names.
- [EXTRA] remove a user from a team.
- [EXTRA] reset the state of the app.
